# Smart Teacher UI Screens

تم إنشاء واجهتين للتطبيق بناءً على التصاميم المقدمة:

## الشاشات المتاحة

### 1. شاشة اختيار نوع الحساب (Account Type Selection)
**الملف:** `lib/features/onboarding/presentation/pages/account_type_selection_page.dart`

**المميزات:**
- اختيار بين مدرس وطالب
- تصميم متجاوب مع دعم RTL
- أيقونات واضحة لكل نوع حساب
- زر متابعة للانتقال للخطوة التالية
- رابط للتسجيل للمستخدمين الجدد

### 2. شاشة المجموعات الدراسية (Study Groups)
**الملف:** `lib/features/groups/presentation/pages/study_groups_page.dart`

**المميزات:**
- شريط بحث عن المجموعات والطلاب
- فلاتر للصفوف الدراسية
- بطاقات المجموعات تحتوي على:
  - صورة المجموعة
  - عنوان المجموعة
  - عدد الطلاب المسجلين
  - الأيام والأوقات
  - حالة المجموعة (نشط/غير نشط)
  - زر لإدارة المجموعة
  - صور الأعضاء
- زر عائم لإضافة مجموعة جديدة
- شريط تنقل سفلي

## كيفية التبديل بين الشاشات

في ملف `lib/main.dart`، قم بتغيير السطر التالي:

```dart
// لعرض شاشة اختيار نوع الحساب
home: const AccountTypeSelectionPage(),

// أو لعرض شاشة المجموعات الدراسية
home: const StudyGroupsPage(),
```

## التشغيل

1. تأكد من تثبيت التبعيات:
```bash
flutter pub get
```

2. شغل التطبيق:
```bash
flutter run
```

## ملاحظات

- جميع الـ Widgets تم تقسيمها إلى components صغيرة قابلة لإعادة الاستخدام
- الكود يتبع معايير Clean Code
- استخدام StatelessWidget قدر الإمكان
- دعم كامل للغة العربية وال RTL
- التصميم يطابق الصور المقدمة
